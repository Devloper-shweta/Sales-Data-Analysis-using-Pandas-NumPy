📊 Sales Data Analysis using Pandas & NumPy
🔹 Project Overview

This project focuses on analyzing sales data to extract meaningful insights using Python libraries like pandas and NumPy.
The dataset contains information about customer orders, products, pricing, discounts, and delivery details.

🔹 Objectives
Perform data cleaning and preprocessing
Handle missing values and inconsistent data
Analyze sales trends and customer behavior
Generate insights to support business decisions
🔹 Key Features

✔️ Data Cleaning (handling null values, formatting)
✔️ Feature Engineering (Total Sales, Final Price after discount)
✔️ Data Analysis (Revenue, top products, regions)
✔️ NumPy-based statistical operations (mean, std, normalization)
✔️ Grouping and aggregation (Region, Category-wise analysis)
✔️ Sorting, filtering, and pivot tables

🔹 Technologies Used
Python 🐍
pandas
NumPy
🔹 Dataset

The dataset includes:

Order details
Customer information
Product category
Pricing and discount
Delivery time
🔹 Sample Insights
Electronics category generated the highest revenue
North region contributed the most sales
Laptops were the top-selling product
